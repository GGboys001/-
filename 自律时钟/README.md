# 自律时钟

一个纯离线的 Android 学习计时与目标打卡工具。应用使用 Kotlin、Jetpack Compose、Room、DataStore 和前台服务实现。

## 打开项目

1. 在 Android Studio 中选择“打开”，定位到本项目根目录。
2. 使用 JDK 17，同步 Gradle 依赖并安装 Android SDK Platform 35。项目已包含 Gradle Wrapper。
3. 连接 Android 设备或启动模拟器后运行 `app`。

## 功能

- 正计时与倒计时，支持暂停、后台持续和通知栏控制。
- 自定义学习任务、每日目标与每周目标。
- 每日任意生效目标达成后自动打卡；连续 7 天奖励消除卡，最多 3 张。
- 按日、周、月和全历史查看任务时长占比、排行及摘要数据。
- JSON 本地导出与导入，导入会覆盖当前本机数据。

## 说明

项目最低支持 Android 8.0（API 26）。首次运行时请允许通知权限，以便后台计时显示常驻通知和倒计时完成提醒。
